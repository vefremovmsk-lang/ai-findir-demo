Стратегические факторы
Корпоративный партнёрский канал — защищённый
>70% крупных сделок — промышленные предприятия в периметре корп. партнёра (Госкорпорация-Х): Клиенты А, Б, В, Г, Д, Е, Ж, З

Конкурентная защита:
  • Конкурент А (нац. лидер) — нет институционального доступа к Госкорпорации-Х
  • Конкурент Б (проектный) — другой отраслевой фокус
  • Никто из конкурентов не работает с этой сетью системно

Риск концентрации: весь канал держится на одном менеджере. Уход = потеря портфеля целиком.

Субхолдинг-Х: закрыт без корпоративного входа
Потери: Клиент-ОДК-А (1 650 000 ₽) + Клиент-ОДК-Б (1 880 000 ₽) = 3 530 000 ₽ суммарно

Паттерн: 2 крупных проигрыша подряд — системный, не случайный.

Гипотезы:
  • Субхолдинг имеет centralized procurement вне доступной корп. сети
  • Конкурент А или Б — preferred vendor у данного субхолдинга
  • Требуется полный пакет с лабораторией → без своей лаборатории проигрываем
Рекомендация: не инвестировать ресурсы в данный субхолдинг без стратегической связи на его уровне. Фокус — Госкорпорация-Х без Субхолдинга-Х — там доступ работает.
Альтернатива: отдельный KAM с корпоративной связью на уровне субхолдинга. Текущий менеджер неэффективен в этом сегменте.

5 ключевых выводов
1
Q2 дефицит = 690К — закрывается одной сделкой. Клиент-П1 (3.5М, 71 день ожидания) — единственный рычаг. Без этой сделки Q2 убыточен вне зависимости от остального пайплайна. Одно совещание Менеджера Б с клиентом = разница между убытком и прибылью квартала.
2
Майская выручка не подтверждена. Аномалия переноса выручки + Исполнитель А 213% плана — Q2 может быть хуже на 1М, чем показывает таблица. Даже с ключевой сделкой 50% при риске переноса Q2 = −210К. Верификация по актам до 16.05 обязательна.
3
Корпоративный партнёрский канал — защищённый, без конкурентов. Регуляторная волна = немедленный спрос. Ни один из верифицированных конкурентов не имеет институционального доступа к данной корп. сети. Новые нормативные требования 2026 создают обязательный спрос у текущих клиентов прямо сейчас. Окно захлопнется к Q4 2026.
4
Q3 pipeline = 10–13М при системной работе. Клиент-П2 (2.75М) + Клиент-С2 (1.47М) + Клиент-П4 (3.32М) + Клиент-П5 (2.55М) + Клиент-П6 (3М) — реалистичная Q3-база при еженедельном касании каждого аккаунта. Без системы — стоять месяцами.
5
Субхолдинг-Х закрыт (3.53М потерь подряд). Два последовательных проигрыша крупных сделок одного субхолдинга — системный паттерн. Субхолдинг имеет отдельную закупочную вертикаль вне доступного корп. канала. Без корпоративного входа на его уровне ресурсы не дают возврата.
